PAINT95 - READ ME ============================== Welcome to Paint95 - Retro Drawing Studio! TOOLS ----- B Brush Freehand drawing with smooth strokes E Eraser Paint background color over strokes F Fill Flood-fill a region (with tolerance) I Pick Eyedropper - sample any pixel color L Line Click and drag for a straight line R Rect Click and drag for a rectangle O Oval Click and drag for an ellipse S Select Drag a selection; Delete to clear it T Text Click canvas, type, Enter to stamp TIPS ---- - Shift+click a palette swatch to set BG color - Right-click a palette swatch to set BG color - Toggle "Filled" for solid vs outline shapes - Toggle "Neon" for a dark canvas with glow effect - Use "2x Zoom" to paint fine pixel detail - Layers are cosmetic labels only (single canvas) - Save stores to browser localStorage - Download exports a full PNG file KEYBOARD SHORTCUTS ------------------ Ctrl+Z Undo Ctrl+Y Redo Ctrl+S Save to My Drawings Ctrl+D Download PNG Delete Clear selection area B/E/F/I/L/R/O/S/T Switch tools [ / ] Decrease / Increase brush size ============================== Paint95 v1.1 - Pure HTML/CSS/JS GitHub Pages ready - no build step needed
